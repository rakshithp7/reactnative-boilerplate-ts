# reactnative-boilerplate-ts

This boilerplate includes the following:

- Expo
- React Native (typescript)
- React Navigation
- Redux Toolkit
- tailwindcss

# Installation

```
npx degit rakshithp7/reactnative-boilerplate-ts myapp
```

```
cd myapp
yarn install
```

or npm

```
cd myapp
npm install
```

# To start the server

```
expo starts
```
